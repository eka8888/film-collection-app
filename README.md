# 🎬 Film Collection App

## 📖 Description

Film Collection is a simple Angular application that displays a list of films.
Users can search for films, view details, and add films to favorites.

This project is built using modern Angular features such as:

* Standalone components
* Signals (no RxJS)
* New control flow (@if, @for)

---

## 🚀 Features

* 🔍 Search films by title
* ⭐ Add / remove films from favorites
* 🎬 Film details page
* 🧭 Breadcrumbs navigation
* 📱 Responsive UI
* ❌ 404 page for unknown routes

---

## 🛠 Technologies

* Angular 20+
* TypeScript (strict mode)
* Angular Signals
* Standalone Components

---

## 📂 Project Structure

```
src/
 ├── app/
 │   ├── core/        # services
 │   ├── features/    # pages & components
 │   ├── shared/      # reusable parts (pipes, directives)
 │   ├── app.routes.ts
 │   ├── app.html
```

---

## ▶️ How to Run

```bash
npm install
ng serve
```

Open your browser at:

http://localhost:4200

---

## 🧠 Key Concepts

* Signals for state management
* Component communication via input() / output()
* Routing with parameters
* Custom directive (autofocus)
* Custom pipe (duration formatter)
